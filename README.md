# Kanna Portfolio

A modern, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- 🎨 **Dark theme** with glassmorphism effects and gradient accents
- ✍️ **Typed text animation** cycling through developer roles
- 🌌 **Animated particle canvas** in the hero section
- 🔄 **Scroll-reveal animations** using Intersection Observer API
- 🧭 **Sticky navbar** with active section highlighting
- 📱 **Fully responsive** for mobile, tablet, and desktop
- 📬 **Contact form** with success feedback
- ✨ **Mouse-follow glow effect** on project cards

## 🗂️ Structure

```
kanna-porfolio/
├── index.html    # Main portfolio page
├── styles.css    # All styles (CSS custom properties, responsive)
├── script.js     # Interactivity (particles, typed text, animations)
└── README.md
```

## 🚀 Getting Started

Just open `index.html` in your browser — no build step required.

To host on **GitHub Pages**:
1. Go to your repo → **Settings** → **Pages**
2. Set source to the `main` branch, `/ (root)` folder
3. Your portfolio will be live at `https://kannanayagam.github.io/kanna-porfolio/`

## 🛠️ Customization

- Update your name, bio, skills, projects, and experience directly in `index.html`
- Swap the avatar SVG placeholder with your real photo
- Add your real email and social links
- Replace the demo project cards with your actual projects
