# VisualFolio

My personal portfolio site. Built with React + Vite.

## Running locally

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173/KANNA-PORFOLIO/`

## Deploying

Push to `main` branch - GitHub Actions handles the rest. Site goes live at:
https://kannanayagam.github.io/kanna-porfolio/

## Stack

- React
- Vite
- Tailwind CSS
